# LumoVerse

Official website source code for LumoVerse.

## Tech Stack
- Laravel
- Blade / Tailwind
- MySQL

## Setup
1. Clone the repo
2. Run `composer install`
3. Copy `.env.example` to `.env`
4. Set DB credentials
5. Run `php artisan key:generate`
6. Run `php artisan migrate`
7. Serve with `php artisan serve`

Production-ready Laravel application.
